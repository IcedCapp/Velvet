        Velvet is a java library being developed by Vincent Galloro (me) in my free time. The idea behind Velvet is to reduce the time spent rewriting the same modules every time a new project is started. When I have an idea, I want to get to work building it, but I often end up spending more time than I'd like recreating the same objects that I've already built in dozens of other projects. My solution to this was Velvet
